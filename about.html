About

Help us build a public list of all the world’s politicians, in a consistent format.

Why?

If data is in a consistent, structured format, it can be reused by developers everywhere. They don’t have waste time scraping data and converting it into a format they can use; instead, they can simply concentrate on making useful tools. And those tools can more easily be picked up, used and adapted to local needs anywhere in the world, saving everyone time and effort.

We’re providing our data in the Popolo standard. That’s a rich, expressive format, but when we add data to EveryPolitician we always use it in the same way. It’s because we’re always consistent about this that the same tools can work with EveryPolitician data from any country, for any country.

And those tools can more easily be picked up, used and adapted to local needs anywhere in the world, saving everyone time and effort.

The data

Although the long term goal is to include every elected official in the world, we’re starting by gathering national-level legislators.

To see what data we have, pick the country and then the specific chamber/house and legislative period. We show you the basic list of who the members were (or, the ones we know of), but usually we also have more information than what’s displayed: contact details, photos, gender, etc. If you want to use that data, you can download it in two useful formats:

  • CSV format (great for spreadsheets)
  • JSON in Popolo format (ideal for developers)

Want more detail? Interested in using this data in a web application or tool you’re building? See the technical overview of EveryPolitician.

What next?

The data can also be used in bigger projects, such as making it easy to track what those politicians say, or visualise their political careers, or for citizens to send public messages to them.

As data becomes available from more countries, all in the same format, more and more tools will become possible.
