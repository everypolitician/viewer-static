Ruby programmer? It’s easy to use this data in your own app! Use the gem like this: 

require 'everypolitician'
country = Everypolitician::Index.new.country('Norway')
house = country.legislature('Storting')
house.popolo.terms.find_by(id: 'term/1973-77').memberships.each do |membership|
  puts "#{membership.person.name} (#{membership.party.name}): #{membership.person.gender}"
end

Libraries also available in Python and PHP.

Main Sources: http://data.stortinget.no/, http://wikidata.org/, http://gender-balance.org/
